# 📘 Live Blog Web App  
**Mata Kuliah: IPBD (Integrasi Platform dan Basis Data)**

Project ini merupakan aplikasi **Live Blog berbasis Web** yang terhubung dengan **REST API (FastAPI)** menggunakan **Fetch API**. Aplikasi ini memungkinkan pengguna untuk melakukan registrasi, serta mengelola blog (create, read, update, delete).

---

## ✨ Fitur Utama
- 🔐 Register User  
- 📝 Create Blog  
- 📖 Read / List Blog  
- ✏️ Edit Blog  
- 🗑️ Delete Blog  
- 🔎 Search Blog  
- 🌙 Dark Mode Toggle  
- 📱 Responsive Design  
- ⚡ Fetch API (tanpa reload halaman)  

---

## 🛠️ Teknologi yang Digunakan
- HTML5  
- Tailwind CSS (CDN)  
- JavaScript (Vanilla JS)  
- Fetch API  
- FastAPI (Backend)
